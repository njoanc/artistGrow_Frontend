# artistGrow

Welcome to artistGrow, a platform to help artists grow their audience and showcase their work!

## Features

- Register as an artist or a fan
- Artists can upload their work and create a portfolio
- Fans can discover and follow artists
- Integrated messaging system for communication between artists and fans

## Technologies Used

- React.js for the frontend
- Node.js and Express.js for the backend
- MongoDB for the database
- GraphQL for API queries and mutations
- Apollo Client for managing GraphQL requests in the frontend
- Material-UI for styling
